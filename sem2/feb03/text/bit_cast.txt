Реализуйте шаблон функций

template<typename T, typename U>
T bit_cast(U);
которые превращают переменную типа U в переменную типа T, сохраняя побитовое представление.

Гарантируется, что sizeof(T) == sizeof(U).



Тип U будет выводиться компилятором автоматически, поэтому функцией можно пользоваться так:

float const f = bit_cast<float>(0x3f800000); // U -> unsigned int, f -> 1.f
Подсказка: взять от переменной типа U указатель, кастануть его в T, разыменовать, вуаля.
